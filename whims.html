Whims!

A few projects outside of UI/UX Design that I'm proud of

Websites

Wordpress Website

This self made website combined my knowledge of Adobe Creative Cloud programs, resulting in a fun journal-styled web project centered around Otome game "Mystic Messenger".

I used Photoshop to create the opening image. The image portrays the nature of Mystic Messenger where each character presents themselves as leading a stable life but the further the player (MC) gets in the game, the more they see how wrong that is.

The next featured project in the website "Mystic Messenger Chats" is a podcast I lead, using Audition to record and edit the audio and using Illustrator to create the background image.

Lastly, I used Premier Pro to create and edit a short video, explaining the Prologue and Day 1 gameplay for Mystic Messenger.

Game Programming

Custom Game Engine

I created this game engine, including a toolbar, hierarchy, editor, inspector, and assets view. The first video shows a demo of how the editor can be used. The user can add assets to the scene, change their scaling value and coordinates of the object, as well as toggle things like gravity and collision.

The second video shows the gameplay inside of the editor. This is a small-scale combat game. The player acts as an alien whose goal is to get back to the portal with all three potions from the lab. The alien must fight off the scientists to get to the potions. Once they return to the portal, they are taken back to their motherland!

This project was made using Visual Studio Code and OpenGL.

Mini Game

This mini game "Edna's Mini Garden" was my first game programming project. I wanted to utilize bright colors with a 'cozy gaming' style to create this short but memorable game.

The object of the game is to collect all seeds and plant them, watching as they rapidly grow to create a beautiful garden!
